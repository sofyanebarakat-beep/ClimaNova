Chaque carte = une image à photographier ou sourcer. Remplacez le fichier sur le serveur une fois la photo prête.

Comment utiliser ce guide :
1. Photographiez ou sourcez chaque image selon les dimensions indiquées.
2. Nommez le fichier exactement comme indiqué (case sensitive).
3. Convertissez en .webp pour les photos (+ garder .jpg en fallback).
4. Uploadez dans /images/ sur le serveur.
5. Le alt text est déjà en place dans le code HTML — rien à modifier.

📷 Conseils photo :
• Fond blanc ou fond de chantier propre — évitez le désordre en arrière-plan
• Techniciens en tenue de travail propre avec logo ClimaNova visible si possible
• Éclairage naturel privilégié — évitez le flash direct
• Photos horizontales (paysage) pour les bannières, carré pour les avatars
• Après la prise de vue : compressez via squoosh.app et convertissez en WebP
• Nommez exactement comme indiqué ci-dessus avant upload
